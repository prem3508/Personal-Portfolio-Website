Personal Portfolio Website
This repository contains my personal portfolio website, created to showcase my skills, projects, and contact information in a clean and modern way. The portfolio is designed with a focus on simplicity, responsiveness, and user-friendly navigation.

About the Project
The Personal Portfolio Website represents my professional profile on the web. It highlights who I am, what I do, and the technologies I work with. The website is suitable for sharing with recruiters, internship providers, and collaborators who want to learn more about my background and work.

The project is built using core web technologies and does not rely on heavy frameworks, making it lightweight and fast.
